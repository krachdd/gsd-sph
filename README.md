# General Simulation Data Format for SPH Solver

Version: 3.4.2

A subproject to the HOOMD SPH Code to keep the [General Simulation Data Packages form Glotzer Group](https://github.com/glotzerlab/gsd) conform with the fields used is SPH. 
For documentation see `gsd`. Modifications are only done to the python api.

## Build 

For the `conda` VENV that is requried for the project see main `hoomd-sph3` project.

```bash
# GSD Package
mkdir build ; cd build 
cmake .. 
make
```
